# 图片转PDF (Pictures-to-PDF)



用于将多张图片快速转换为PDF文件，支持合并为单个PDF或每张图片生成独立PDF，操作简单直观。


## 功能特点



- **可视化操作**：简洁的 GUI 界面，无需命令行操作，轻松上手。

- **核心功能**：

  - 支持选择包含图片的文件夹（支持JPG、PNG格式）
  
  - 两种转换模式：所有图片合并为单个PDF / 每张图片生成独立PDF

  - 实时显示转换进度，支持日志查看

  - 图片自动缩放并居中显示在PDF页面中

  - 转换完成后生成统计信息：总图片数、成功转换数、耗时

  - 支持自定义PDF页面尺寸（A4/Letter等）

  - 一键清空历史记录，方便重复操作



## 环境要求



- Python 3.6 及以上

- 依赖库：

    - Pillow（图片处理）

    - reportlab（PDF生成）


## 快速开始



1. **克隆仓库**

    ```Bash
    git clone https://github.com/DestinyQwQ/Pictures-to-PDF.git
    cd Pictures-to-PDF
    ```

2. **运行程序**

    直接执行 Python 脚本：

     ```Bash
      python main.py
     ```



1. **使用步骤**

    1. 点击「选择图片文件夹」按钮，导入需要转换的图片

    2. 选择转换模式（合并为单个PDF / 生成多个PDF）

    3. 点击「设置输出路径」，指定PDF保存位置

    4. 点击「开始转换」，等待处理完成

    5. 在「转换结果」区域查看：

        - 转换日志：详细过程记录

        - 统计信息：转换数量及耗时

    6. （可选）点击「打开输出目录」直接查看生成的PDF；点击「清空列表」可重新导入图片


## 开发技术栈



- 界面框架：Tkinter

- 图片处理：Pillow (PIL)

- PDF生成：reportlab

- 核心逻辑：文件遍历 + 格式转换

- 多线程：避免界面卡顿



## 注意事项



1. 目前支持的图片格式：JPG（.jpg, .jpeg）、PNG（.png），其他格式可能导致转换失败

2. 大尺寸图片转换可能占用较多内存，建议分批处理

3. 转换后的PDF默认使用A4页面尺寸，可在「设置」中修改

4. 请确保图片文件路径不包含中文或特殊字符，避免读取错误
